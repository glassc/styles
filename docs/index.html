Personal Style Guide

A clean, modern design system with light & dark modes

Attribution

This style guide uses the Nord color palette for both light and dark themes, with layout and typography patterns inspired by Ghost's Journal theme. Design elements have been adapted for use with Astro and GitHub Pages.

Usage Examples

Integration examples for Astro and GitHub Pages.

Astro Integration

--- // src/layouts/Layout.astro --- <html lang="en"> <head> <link rel="stylesheet" href="/styles.css"> </head> <body> <div class="container"> <slot /> </div> </body> </html>

GitHub Pages Integration

<!-- Add to your GitHub Pages _layouts/default.html --> <!DOCTYPE html> <html lang="en"> <head> <link rel="stylesheet" href="{{ '/assets/css/styles.css' | relative_url }}"> </head> <body> <div class="container"> {{ content }} </div> </body> </html>
